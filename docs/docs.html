Informations
Introduction
The Discord ZT API was created on July 23, 2023 (the first version was usable on July 26th of 2023).
The purpose of this API is to allow third-party applications to easily retrieve information from the Zone-telechargement site without having to do scraping.


Why this API ?
The purpose of this API is to allow third-party applications to easily retrieve information from the Zone-telechargement site without having to do scraping.

How to use it ?
Its really simple. With your API token you make a request to our API asking what do you want, its entierely customizable..

Get early access
Unfortunatly the API is still in a pre-beta version, meaning that its not public.
But keep in mind that if you're using this API in pre-beta / beta / alpha / canary and any other unstable versions, you'll have to often check our documention to see if there is no modifications.
You can find the versions changelogs in the left-side menu.

GET /api/search
Endpoint to search movies by page

Query parameters
FIELD
TYPE
DESCRIPTION

string
The category for the search (films, series, ...)
query
string
The string query

Simple getMovieDetails query

GET /api/getMovieDetails
Endpoint to get movie details

Query parameters
FIELD
TYPE
DESCRIPTION
id
string
The ID of the element/films/whatever you want details. ID can be known with search endpoints
?param Optional parameter

The ID is 8726, so in our request, we will request that ID.

2
